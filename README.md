# Silver Proxy

Este aplicativo encaminha o tráfego dos seus serviços pela conexão do seu celular, criando um proxy mobile dedicado.

## O mesmo IP do seu celular

Suas requisições usarão a mesma conexão de internet do aparelho, aumentando a consistência e a confiabilidade.

## Grande economia

Serviços de proxy mobile de qualidade custam caro. Oferecemos essa facilidade de graça para nossos clientes, para que você possa investir no que realmente importa.

## Fácil de Usar

Basta adicionar sua instância e conectar. O aplicativo cuida de todo o resto, garantindo uma conexão estável e segura.

## Dica

Mantenha o app configurado para ficar sempre aberto e fora das otimizações de bateria/hibernação, evitando perder a conexão.
